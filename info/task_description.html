This is the first in a series of missions dedicated to cryptography. In this mission we will take a look at the Atbash cipher.

Atbash is one of the oldest known ciphers, created in Middle East around 600 B.C; the first known examples of it can be found in Tanakh (the Old Testament), where it was used to encrypt certain names and words.

Atbash is very simple substitution cipher where the key is formed by mapping the plaintext alphabet to it's reverse, so that the first letter is replaced with the last, the second letter - with the second to last and so on. For the Latin alphabet the substitution table looks like this:

Plaintext:  abcdefghijklmnopqrstuvwxyz
Ciphertext: ZYXWVUTSRQPONMLKJIHGFEDCBA

Of course, since the encryption algorithm doesn't use any type of secret key, Atbash provides virtually no information security. Anyone who intercepts the message encrypted with the Atbash cipher can easily decipher it by using the same substitution table.

The word "Atbash" is composed of hebrew letters Aleph, Tae, Bet and Shin and actually describes the substitution principle: Aleph (first letter of hebrew alphabet) to Tae (last one), Bet (second) to Shin (second to last), etc.

For this mission you need to write a function that encrypts given text with Atbash. Punctuation, whitespaces and other characters should remain unchanged.

Input: Plaintext: str

Output: Ciphertext: str
